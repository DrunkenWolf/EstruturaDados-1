1) [2,0] Assinale (V) para indicar que a afirmação é VERDADEIRA e (F) para FALSA. Se verdadeira, apenas
reescreva a frase na folha de respostas; mas, se falsa, reescreva a sentença tornando-a verdadeira.
OBS: A cada duas erradas, anula-se uma certa. E, se assinalar (F) e não justificar corretamente, será
considerado ERRADA a alternativa.
a. Toda lista linear possui a ordem lógica dos elementos (a ordem ‘vista’ pelo usuário) sendo a mesma ordem física (em
memória principal).
b. Na estrutura de dados do tipo lista linear ordenada, todo elemento novo que é inserido torna-se o primeiro elemento
da lista.
c. Lista linear ligada, utilizando implementação estática, possui o procedimento de alocação/desalocação de memória
para os registros sob demanda.
d. Lista linear ligada duplamente encadeada possui o procedimento de alocação/desalocação de memória para os
registros sob demanda.
e. Em uma estrutura de lista, o último elemento armazenado deve ser o primeiro a ser removido.

Respostas:
a)[F] Nem toda lista linear possui a ordem lógica dos elementos (a ordem ‘vista’ pelo usuário) sendo a mesma ordem física (em
memória principal), visto que é possível citar a lista linear ligada estática como um contraexemplo.
b)[F] Na estrutura de dados do tipo lista linear ordenada, todo elemento novo que é inserido assume uma 
posição que é determinada a partir da comparação do novo elemento com os demais já inseridos na lista.
c)[F] Lista linear ligada, utilizando implementação estática, não possui o procedimento de alocação/desalocação de memória
para os registros sob demanda, pois isso é uma característica das listas com implementação dinâmica.
d)[V] Lista linear ligada duplamente encadeada possui o procedimento de alocação/desalocação de memória para os
registros sob demanda.
e)[F] Em uma estrutura de lista, não é necessário que o último elemento armazenado seja o primeiro a ser
removido, pois o procedimento de exclusão pode atuar sobre qualquer posição da lista.
